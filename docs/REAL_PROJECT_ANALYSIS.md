## 6. Key Findings

### 6.1 What We Learned

1. **Real projects have 2 bathrooms per unit** (at least for 3+1/4+1)
2. **Interior finishes alone can cost 260,000-415,000 TL/unit** (luxury level)
3. **74 light fixtures per unit** is typical for luxury
4. **Premium brands** (Grohe, Duravit, Franke) are common in mid-high projects
5. **Detailed material lists exist** - contractors track every item

### 6.2 What's Missing from Sample Data

1. Rough construction costs (concrete, steel, labor)
2. MEP installation costs (only fixtures, not pipes/wires)
3. Land cost
4. Unit sizes (NET m² per unit type)
5. Sale prices
6. Timeline

### 6.3 Recommendations for Calculator

1. **Add bathroom count** as a variable (1-3 per unit)
2. **Track interior finish quality** by brand tier (Economy/Standard/Premium/Luxury)
3. **Use quantity-based calculations** where possible (lights per unit, fixtures per bathroom)
4. **Separate kitchen costs** from bathroom costs
5. **Add common area costs** as % of project
